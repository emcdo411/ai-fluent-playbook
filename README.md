## Versioning

This playbook uses a capability-aware versioning model.
See [CHANGELOG.md](CHANGELOG.md) for the full version history and capability review log.

**Review cadence:** Every 6 months, or at any major AI model generation release.

To log a capability review:
```powershell
.\tools\version-bump.ps1 -CapabilityReviewOnly -AIGeneration "Q3 2026 models" -Notes "Appendix B refresh"
```
