# Playbook Working Directory

This directory is for working drafts, revision tracking, and in-progress edits.

## How to Use

1. Place your working .docx or .md drafts here when implementing changes
2. Use revision-checklist.md to track progress against audit priorities
3. When a revision is complete, update CHANGELOG.md under [Unreleased]
4. Do not commit final publication-ready versions here - this is a working directory

## Version Naming

| Version | Meaning |
|---------|---------|
| v1.0 | Initial publication (March 2026) |
| v1.1 | First post-publication update (P1-P3 + v3 additions implemented) |
| v1.x | Capability updates (Appendix B refresh, governance criteria review) |
| v2.0 | Major structural addition or full capability re-calibration |
